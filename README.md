ResetSign

リセット看板
アイテムを初期化します

----- config.yml -----

・gameMode:
リセット看板を動かすゲームモードにtrueを設定してください

・playerLogMode:
true にするとプレイヤーチャットに動作内容を送信します

----------------------

------ command -------

・resetSign
1.リセット対象にしたいアイテムをメインハンドに持たせる
2.張り付けてある看板をターゲットにしてコマンドを実行してください
3.その後、その看板はメインハンドに持っていたアイテムを持って、
　右クリックした時にリセットされます。
※ シュルカーボックスの場合、中身のアイテムが消えます。

・configReload
config.yml を再読み込みします

----------------------
